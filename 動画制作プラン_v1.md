# Higgsfield 動画制作プラン v1

## 制作テーマ: AI研修・コンテンツ用動画素材セット

### 目標
Higgsfield AI を使って以下の3種類の動画素材を制作し、`outputs/` に格納する。
AI研修教材・SNS・プレゼン資料への即時転用を想定。

---

## 制作リスト

### セット1: AI講師キャラクター（縦型 9:16）
- 用途: AI研修動画のオープニング・ナレーション場面
- 形式: MP4 9:16 / 5〜8秒
- モデル: Soul（画像）→ Seedance（動画）

| # | 場面 | プロンプト方針 |
|---|------|-------------|
| 1-A | 正面向き・話しかけ | 親しみやすい日本人講師、スーツ、白背景 |
| 1-B | ノートPC操作中 | モダンなオフィス、集中した表情 |

---

### セット2: AI・テクノロジーの映像 B-roll（横型 16:9）
- 用途: プレゼン・研修スライドのバックグラウンド映像
- 形式: MP4 16:9 / 8〜10秒
- モデル: Flux（画像）→ Seedance or Kling（動画）

| # | 場面 | プロンプト方針 |
|---|------|-------------|
| 2-A | 近未来オフィス | 明るい空間、AIツールを使う人々、自然光 |
| 2-B | データ可視化 | 光るグラフ・ネットワーク図、青系、抽象的 |
| 2-C | チーム協働 | 多様なメンバーが集まって話し合う、活気ある |

---

### セット3: SNS用ショート動画（縦型 9:16）
- 用途: X(Twitter)・Instagram Reels 投稿素材
- 形式: MP4 9:16 / 5〜6秒
- モデル: Seedream（高速）→ Minimax Hailuо（コスト効率）

| # | 場面 | プロンプト方針 |
|---|------|-------------|
| 3-A | AIツール操作 | スマートフォン操作、動的な光エフェクト |
| 3-B | ビフォー/アフター | 変化・進化を表す映像的メタファー |

---

## 実行順序（コスト最小化）

```
Step 1: ドラフト確認フェーズ（低コストモデルで内容確認）
  → Seedream + Minimax Hailuo でまず全シーンのドラフトを生成
  → outputs/01_画像/draft/ と outputs/02_動画/draft/ に格納

Step 2: 本番生成フェーズ（合格したシーンを高品質モデルで再生成）
  → Soul / Flux + Seedance / Kling で最終品を生成
  → outputs/01_画像/final/ と outputs/02_動画/final/ に格納

Step 3: サムネイル抽出
  → 動画の1フレームをサムネイルとして outputs/03_サムネイル/ に保存
```

## 代替案: fal.ai で即時実行

Higgsfield のクレジット待ち期間中は **fal.ai** で同等の生成が可能。

**セットアップ**:
```bash
# fal.ai APIキー取得: https://fal.ai → ダッシュボード → Keys
export FAL_KEY="your-fal-key-here"
# ~/.claude/settings.json の env セクションに追加することで永続化
```

**同等モデル対応表**:
| Higgsfield | fal.ai 代替 |
|-----------|------------|
| Soul V2（画像） | `fal-ai/nano-banana-pro` |
| Flux（画像） | `fal-ai/flux/dev` |
| Seedance（動画） | `fal-ai/seedance-1-0-pro` |
| Kling（動画） | `fal-ai/kling-video/v3/pro` |

fal.ai は新規登録で $2.50 の無料クレジット付き。制作プランv1のドラフト分（~$1〜2相当）をカバー可能。

## 次のアクション（優先順）

### Option A: Higgsfield プランを契約して実行
1. [higgsfield.ai/pricing](https://higgsfield.ai/pricing) で **PLUSプラン**を契約（$39/mo・年払い）
2. Claude Code を再起動 → `higgsfield` MCP が接続される
3. `セット2-A`（Flux画像→Seedance動画）からドラフト生成を開始
4. `outputs/01_画像/draft/` → `outputs/02_動画/draft/` に格納

### Option B: fal.ai で即時実行（APIキー取得後）
1. [fal.ai](https://fal.ai) で無料登録 → APIキーを取得
2. `export FAL_KEY="..."` を設定
3. `/fal-ai-media` スキルを使って同じプロンプトで生成開始
4. 出力を同じ `outputs/` フォルダに保存
