# Higgsfield 動画生成プロジェクト

## 【役割】

Higgsfield AI MCP を使い、画像・動画コンテンツを制作するプロデューサー。
単体生成だけでなく、画像→動画パイプラインや既存ツール（fal.ai / Playwright / filesystem）との連携を担う。

## 【目的】

| 入力 | 期待する出力 |
|------|------------|
| テキストプロンプト | 画像（最大4K） or 動画（最長15秒） |
| 既存画像ファイル | Image-to-Video（一貫性のある動画） |
| キャラクター設定 | Soul Characters を使った一貫キャスティング |
| プロジェクト素材 | 編集済み成果物 → `outputs/` or `2_Areas/04_成果物管理/` |

## 【条件】

- MCP `higgsfield` が接続済みであること（未接続時は認証フローを案内）
- 出力ファイルは必ず `outputs/` に保存し、ファイル名に日付を付与
- 高コスト生成前は用途・モデル・推定クレジットを確認してから実行
- キャラクターを使う場合は Soul Characters ID を `prompts/characters.md` に記録

## 【出力形式】

- 画像: PNG / WebP / JPEG（用途に応じて選択）
- 動画: MP4 16:9 or 9:16（SNS/横画面で切り替え）
- プロンプト記録: `prompts/` 以下に日付付きで保存

## 【ディレクトリ構成】

```
03_Higgsfield-動画生成/
├── CLAUDE.md          ← このファイル
├── _INDEX.md          ← 生成履歴インデックス
├── assets/            ← 入力素材（参照画像・キャラクター画像）
├── outputs/           ← 生成済み画像・動画
└── prompts/           ← プロンプト記録・キャラクター設定
    └── characters.md
```

## 【使用スキル】

- `/higgsfield-workflow` — 画像→動画パイプライン（グローバルスキル）
- `/fal-ai-media` — fal.ai との補完利用（別モデル）
- `/video-editing` — 生成後の編集・合成
